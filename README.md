# AI_Recommendation_System

## Overview
This project is a simple AI Recommendation System developed in Python. It recommends items based on the user's selected interest category.

## Features
- Takes user input (interest/category)
- Matches preferences using conditional logic
- Displays personalized recommendations
- Beginner-friendly AI project

## Technologies Used
- Python

## Categories
- Movies
- Books
- Music
- Sports
- Technology
